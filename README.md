# Personal Portfolio Website

My personal portfolio website showcasing my projects and skills.

## Development Setup

### Prerequisites
- Python 3.8 or higher
- Poetry

### Installation
```bash
# Install dependencies
poetry install

# Start development server
poetry run python -m http.server 8080
```

### Project Structure
```
web/
├── static/
│   ├── css/
│   ├── js/
│   └── img/
├── index.html
├── pyproject.toml
└── README.md
```
